# Cosmetic Pop-Ups Sales Analysis with SQL
## Overview
This project analyzes cosmetic pop-up store sales data using SQL to uncover customer purchasing behavior, product performance, and business insights. 

## Objectives
- Analyze sales performance across products
- Identify top-selling categories and items
- Understand customer purchasing patterns

## 🛠️ Tools & Technologies
- SQL
- MySQL
- MySQL Workbench

## 📂 Dataset
The dataset contains cosmetic pop-up store transaction data, including
- Customer information
- Product details
- Categories
- Quantity purchased
- Prices
- Sales transactions
- Purchase dates

## SQL Skills Demonstrated
- SELECT statements
- Filtering using WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions
  - COUNT()
  - SUM()
  - AVG()
  - MAX()
  - MIN()
- JOINs
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (if applicable)
- CASE WHEN
- Date Functions

## 📈 Example Business Questions
Some of the questions answered include:
- Which products generate the highest revenue?
- Which product categories sell the most?
- What is the average customer spending?
- Who are the highest-value customers?
- Which days or months record the highest sales?
- Which products have the lowest sales performance?
